## どこまで読んだか
多分150ページまで流し読みした。

コーダーの人は買っておいてもいいと思う。
こんなんもCSSだけでできるのか！？と驚かされた一冊。

CSSは簡単だと思われがちだが、いろいろ知るとめちゃくちゃ難しいことがわかる。
実務でここまで必要かといわれると必要ないことがほとんどだし、ここまでCSSで書ける人は少ないと思う。CSSだけでやるとどうしてもわかりにくい部分はあるので、SASSで一部分変数化してコメントを入れるなどすると管理しやすいCSSになると思う。

ちょっとテクりたいなと思ったらまた読もうと思う。
