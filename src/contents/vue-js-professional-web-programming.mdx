## 感想
Vue関連の本はよさげなものはとりあえず買うことにしている。

どこまで読んだか忘れたが、モーダルやアコーディオンなどサイト制作でよく使うコンポーネントの作り方が書いてあるのでこれを一通りやれば大体のコンポーネントの作り方はなんとなく把握できると思う。

## microCMS
なんと最後らへんのページでmicroCMSから記事を取得してNetlifyにデプロイしたりWebhookする方法も書かれている。

こういう本は基礎的な構文の説明やよく使うライブラリ周りの説明で終わってしまうものも多い中、具体的な例がたくさん載っているので実用的な本だと思う。
